# Dhan Trading Dashboard

A self-hosted trading dashboard for the [DhanHQ](https://dhanhq.co/) platform. Single Docker image that runs a FastAPI dashboard, JupyterLab for market analysis, and Caddy reverse proxy with automatic TLS.

## Quick Start

```bash
# 1. Clone and configure
git clone https://github.com/athcodes2005/dhan-python.git
cd dhan-python
python generate_env.py   # interactive credential setup

# 2. Run
docker compose up -d
```

The dashboard will be available at `http://localhost` (or your configured domain with HTTPS).

## Docker Run

```bash
docker run -d \
  --name dhan-trading \
  --restart unless-stopped \
  -p 80:80 -p 443:443 \
  --env-file .env \
  -e DOMAIN=localhost \
  -v config:/config \
  -v data:/app/data \
  -v notebooks:/app/notebooks \
  athcodes2005/dhan-python:latest
```

## What's Inside

| Service | Description | Access |
|---------|-------------|--------|
| **Dashboard** | Portfolio overview, token management, instrument search | `/` |
| **JupyterLab** | Market data analysis with pandas, plotly, pandas-ta | `/lab/` |
| **Caddy** | Reverse proxy with automatic HTTPS | Ports 80/443 |

## Dashboard Pages

- **Home** — Fund limits and balance metrics
- **Authentication** — Token status, IP whitelisting, Data API status, auto-renewal toggle
- **Account** — Holdings and positions with portfolio history tracking
- **Search** — Instrument search across NSE/BSE exchanges
- **Lab** — JupyterLab environment for data analysis notebooks

## Login

| Username | Role | Access |
|----------|------|--------|
| `admin` | Admin | Full access: generate tokens, manage settings |
| `guest` | Guest | View only: balances, holdings, positions, search |

Passwords are configured in `.env` (`ADMIN_PASSWORD`, `GUEST_PASSWORD`).

## Environment Variables

Create a `.env` file manually or use `python generate_env.py`:

| Variable | Required | Description |
|----------|----------|-------------|
| `DHAN_CLIENT_ID` | Yes | Dhan client ID |
| `DHAN_API_KEY` | Yes | Dhan API key |
| `DHAN_API_SECRET` | Yes | Dhan API secret |
| `DHAN_PIN` | Yes | 6-digit trading PIN |
| `DHAN_TOTP_SEED` | Yes | TOTP seed for 2FA |
| `SECRET_KEY` | Yes | Session signing key (auto-generated by `generate_env.py`) |
| `ADMIN_PASSWORD` | Yes | Dashboard admin password |
| `GUEST_PASSWORD` | Yes | Dashboard guest password |
| `DOMAIN` | No | Domain for auto-TLS (default: `localhost` for HTTP) |
| `STATIC_IP` | No | Server IP for Dhan order execution whitelisting |
| `TZ` | No | Timezone (default: `Asia/Kolkata`) |

## Volumes

| Path | Purpose |
|------|---------|
| `/config` | `config.json` — access tokens, settings |
| `/app/data` | Instruments database, portfolio history |
| `/app/notebooks` | JupyterLab notebooks |

## TLS / HTTPS

- **Local development**: Set `DOMAIN=localhost` — serves HTTP on port 80
- **Production**: Set `DOMAIN=your.domain.com` — Caddy auto-provisions Let's Encrypt certificates (requires ports 80/443 open and DNS pointing to your server)
